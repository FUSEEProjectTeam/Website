Class AttributeLocations

Declares buffer locations. OpenGL specific error: GL_INVALID_VALUE is generated if index is greater than or equal to GL_MAX_VERTEX_ATTRIBS. GL_MAX_VERTEX_ATTRIBS seems to be 16.
