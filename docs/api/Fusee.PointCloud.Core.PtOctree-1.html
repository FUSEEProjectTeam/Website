Class PtOctree<TPoint>

Tree data structure in which each internal node has up to eight children. Octrees are most often used to partition a three-dimensional space by recursively subdividing it into eight octants. See OctreeD<P> This subclass is used for point cloud rendering. It has the additional fields MaxNoOfPointsInBucket and PtAccessor.
