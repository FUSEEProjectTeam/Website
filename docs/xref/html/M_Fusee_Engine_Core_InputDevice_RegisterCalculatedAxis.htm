InputDeviceRegisterCalculatedAxis Method

Registers a calculated axis. Calculated axes behave like axes exposed by the underlying hardware device but can be calculated from one or more existing axes or buttons.

Remarks
To register your own axis you need to provide a working InputDeviceAxisValueCalculator. This method is called whenever the axis value needs to be present.
