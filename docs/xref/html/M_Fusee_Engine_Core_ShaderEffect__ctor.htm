effectPasses
Type: Fusee.Engine.CoreEffectPassDeclaration
The ordered array of EffectPassDeclaration items. The first item in the array is the first pass applied to rendered geometry, and so on.
effectParameters
Type: System.Collections.GenericIEnumerableEffectParameterDeclaration
A list of (uniform) parameters possibliy occurring in one of the shaders in the various passes. Each array entry consists of the parameter's name and its initial value. The concrete type of the object also indicates the parameter's type.
Remarks
Make sure to list any parameter in any of the different passes' shaders you want to change later on in the effectParameters list. Shaders must not contain paramaeters with names listed in the effectParameters but declared with different types than those of the respective default values given here.
